# Agora Partner Portal - Dummy/Demo Version

This repository contains a stripped-down, demonstration version of the Agora Partner Portal. It is intended to showcase the core functionality of the platform, including referral tracking, user dashboards, and role-based access for CPAs and Financial Professionals.

**Note:** This is *not* the production version of the application. 

### Key Differences from Production:
- **No Payment Processing:** Stripe integration and payout logic have been removed from this version.
- **Mocked External Integrations:** Automated verifications (like NASBA or FINRA BrokerCheck) may be mocked or stripped down to prevent unnecessary API usage.
- **Dummy Data:** Any database data associated with this version should be considered dummy data used strictly for demonstration purposes.

The complete, production-ready version of this platform is maintained in a separate, private repository to securely handle payment information and sensitive professional details.
